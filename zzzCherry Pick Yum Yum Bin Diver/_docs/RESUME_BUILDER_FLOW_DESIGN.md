## ğŸ¯ SCORING SYSTEM (Path 1 Only)

### Real Measurable Metrics

#### 1. ATS Score (0-100)
**What it measures**: Applicant Tracking System compatibility

**Factors**:
- Keyword density (BPO-relevant terms)
- Proper section headers present
- File format (PDF preferred)
- Font readability (standard fonts)
- Contact information completeness
- No tables/graphics (ATS can't read them)

**Calculation**:
```typescript
atsScore = (
  keywordDensity * 0.3 +      // 30% weight
  sectionHeaders * 0.2 +       // 20% weight
  formatScore * 0.2 +          // 20% weight
  fontScore * 0.15 +           // 15% weight
  contactInfo * 0.15           // 15% weight
)
```

#### 2. Content Quality Score (0-100)
**What it measures**: Quality of writing and achievements

**Factors**:
- Action verbs usage (started, managed, achieved)
- Quantifiable achievements (numbers, percentages)
- Grammar and spelling
- Relevance to BPO industry
- Professional language (no slang)
- Achievement-focused vs duty-focused

**Calculation**:
```typescript
contentScore = (
  actionVerbs * 0.25 +         // 25% weight
  quantifiableResults * 0.30 + // 30% weight
  grammarSpelling * 0.20 +     // 20% weight
  relevance * 0.15 +           // 15% weight
  professionalism * 0.10       // 10% weight
)
```

#### 3. Formatting Score (0-100)
**What it measures**: Visual presentation and structure

**Factors**:
- Visual hierarchy (clear sections)
- White space usage (not cluttered)
- Consistent styling (fonts, sizes, spacing)
- Length (1-2 pages ideal)
- Professional appearance
- Easy to scan

**Calculation**:
```typescript
formatScore = (
  visualHierarchy * 0.25 +     // 25% weight
  whiteSpace * 0.20 +          // 20% weight
  consistency * 0.25 +         // 25% weight
  lengthOptimal * 0.15 +       // 15% weight
  scannability * 0.15          // 15% weight
)
```

#### 4. Skills Match Score (0-100)
**What it measures**: BPO-relevant skills present

**Factors**:
- BPO keywords (customer service, call handling, etc.)
- Technical skills (CRM, Excel, typing speed)
- Soft skills (communication, problem-solving)
- Certifications mentioned
- Language proficiency stated

**Calculation**:
```typescript
skillsScore = (
  bpoKeywords * 0.35 +         // 35% weight
  technicalSkills * 0.25 +     // 25% weight
  softSkills * 0.20 +          // 20% weight
  certifications * 0.10 +      // 10% weight
  language * 0.10              // 10% weight
)
```

#### 5. Overall Score
**Calculation**:
```typescript
overallScore = (
  atsScore * 0.30 +            // 30% weight - most important!
  contentScore * 0.30 +        // 30% weight
  formatScore * 0.20 +         // 20% weight
  skillsScore * 0.20           // 20% weight
)
```

### Ranking System

**How it works**:
1. Compare total score against database of all analyzed resumes
2. Position = number of resumes with higher score + 1
